## How to use

ライブラリをインストール
```
pip install django Pillow pyaudio
```

djangoページを起動

```
cd audio_stream/stt_with_brouser
python manage.py runserver
```

djangoページへアクセス

startボタンを押下

テキスト化したい音源を再生
(マイクを用いる場合はマイクの集音を開始)
